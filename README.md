Self-Driving-RC-Car
===================

Testing out Kalman filter and SLAM code on a vehicle which can't fall out of the sky.

**Hardware**

IMX (Imex) Firefox 1/10 scale RC car contains one RC550 brushed hobby DC motor and one proprietary unmarked five-wire servo.
